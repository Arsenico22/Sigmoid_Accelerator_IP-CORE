# Design e caratterizzazione di IP-Core su FPGA per funzioni di attivazione in modelli di AI

Questo repository contiene il materiale didattico e il codice sorgente relativo al lavoro di Tesi di Laurea Magistrale in Ingegneria Elettronica (Università della Calabria).

## Struttura del Repository

File sorgenti VHDL delle release Baseline, V2, V3 e V3.1 dell'IP-Core per il calcolo della Sigmoide (.vhd), file necessari per i testbench (.mem) e file di vincolo (.xdc).

Codici sorgente script Python e Matlab utilizzati per la generazione dei vettori di stimolo, la quantizzazione fixed-point (Q1.15) e l'analisi comparativa delle prestazioni.
